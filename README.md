🧩 Overview

The Text-to-Speech Converter is a Python-based application that transforms textual input into audio output. It is useful for accessibility, automation, and voice-enabled applications.

💡 Problem Statement

To build a system that converts user-provided text into human-like speech efficiently, enabling better accessibility and interaction with digital systems.

📊 Dataset & Inputs

No external dataset is required.

Input: User-provided text
Output: Generated speech (audio file or playback)
🛠 Tools & Technologies
Language: Python
Libraries: gTTS / pyttsx3 (update based on your project)
IDE: VS Code / PyCharm
Version Control: Git & GitHub

▶️ How to Run This Project
Clone the repository:
git clone https://github.com/harshitsr04/Text-to-speech.git
cd Text-to-speech
Install dependencies:
pip install -r requirements.txt
Run the project:
python main.py
Enter text and listen to the generated speech 🎧
